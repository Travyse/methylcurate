I can help you curate DNA methylation datasets from GEO and evaluate epigenetic aging clocks using MethylCurate.

You can ask me to help with tasks such as:

- **Retrieve GEO datasets**  
  Example: “Download and process GSE109627.”

- **Extract sample metadata**  
  Example: “Extract age, sex, tissue, diagnosis, and sample IDs from this GEO dataset.”

- **Harmonize metadata fields**  
  Example: “Standardize the tissue and disease labels for this dataset.”

- **Parse DNA methylation supplementary files**  
  Example: “Find the processed DNA methylation beta matrix and format it for analysis.”

- **Run quality control**  
  Example: “Check whether the methylation values are beta values or M-values and apply QC.”

- **Build standardized outputs**  
  Example: “Create a harmonized metadata table and formatted beta matrix for this study.”

- **Evaluate epigenetic aging clocks**  
  Example: “Run aging clock benchmarking on this curated dataset.”

- **Generate logs, artifacts, and visualizations**  
  Example: “Show me the outputs from the current workflow.”

A typical workflow is:

1. Tell me which GEO accession you want to curate.
2. I retrieve the GEO metadata and supplementary files.
3. I help identify and parse the DNA methylation data file.
4. I extract and harmonize sample-level metadata.
5. I construct a standardized beta matrix.
6. I run quality control.
7. I evaluate selected epigenetic aging clocks and return results, artifacts, and visualizations.
