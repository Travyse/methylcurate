# Installation

## Stable release

To install methylcurate, run this command in your terminal:

```sh
uv add methylcurate
```

Or if you prefer to use `pip`:

```sh
pip install methylcurate
```

## From source

The source files for methylcurate can be downloaded from the [Github repo](https://github.com/travyse/methylcurate).

You can either clone the public repository:

```sh
git clone https://github.com/travyse/methylcurate
```

Or download the [tarball](https://github.com/travyse/methylcurate/tarball/main):

```sh
curl -OJL https://github.com/travyse/methylcurate/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd methylcurate
uv sync
```
